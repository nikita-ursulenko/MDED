Инструкция по управлению сайтом "MDED"

Это руководство поможет вам вносить изменения в структуру и контент вашего сайта!

1. Управление меню и навигацией |Nav|

Навигационное меню позволяет пользователям быстро переходить к ключевым разделам сайта. Следующая инструкция объясняет, как добавить новую ссылку в основное меню.

Важное замечание:

В этом меню можно добавить два типа ссылок:

  • Обычная ссылка на внешнюю или внутреннюю страницу.
  • Якорная ссылка на существующий раздел на главной странице. Для этого программист должен предварительно добавить уникальный id к нужному блоку. Например, для блока "FAQ" id может быть #faq, и именно его нужно будет указать при добавлении ссылки.

Как добавить новую ссылку в меню

Шаг 1. Откройте настройки кастомизации.

На главной странице сайта нажмите на кнопку "Customize" (Кастомизировать).

Шаг 2. Перейдите в раздел "Menus".

В боковом меню выберите пункт "Menus" (Меню).

Шаг 3. Выберите "Main Menu".

Выберите "Main Menu" (Главное меню) для редактирования.

Шаг 4. Добавьте новый элемент.

В открывшемся списке текущих пунктов меню нажмите кнопку "Add Items" (Добавить элементы).

Шаг 5. Выберите "Custom Links".

Нажмите на раскрывающийся блок "Custom Links" (Пользовательские ссылки).

Шаг 6. Заполните поля.

В поле "URL" введите полный адрес страницы или якорный ID (например, #my-new-section). В поле "Link Text" введите название, которое будет отображаться в меню (например, "Новый раздел").

Шаг 7. Опубликуйте изменения.

После добавления нового элемента нажмите кнопку "Publish" (Опубликовать) в верхней части экрана, чтобы сохранить изменения и сделать их видимыми на сайте.

2. Главная сеция |Hero|

Смена фотографий слайда

Шаг 1.1. Выберите пункт "Slide-uri".

Для смены фотографий слайдов в главной секции Hero выберите пункт "Slide-uri" (Слайды) в боковом меню.

Шаг 1.2. Просмотр существующих слайдов.

Здесь вы видите уже добавленные фотографии, которые чередуются между собой на сайте в виде слайдов.

Шаг 1.3. Добавление нового слайда.

Чтобы добавить новый слайд, нажмите кнопку "Adaugă un slide nou" (Добавить новый слайд).

Шаг 1.4. Дайте название фотографии.

Дайте название новому слайду. Это поможет вам его идентифицировать в дальнейшем.

Шаг 1.5. Нажмите "Add Image".

Нажмите кнопку "Add Image" (Добавить изображение), чтобы выбрать или загрузить фотографию.

Шаг 1.6. Выбор существующей фотографии.

В открывшемся окне вы можете выбрать уже существующую фотографию из вашей медиатеки. Выберите нужную и нажмите "Select" (Выбрать).

Шаг 1.7. Загрузка новой фотографии.

Чтобы загрузить новую фотографию, перейдите во вкладку "Upload files" (Загрузить файлы). Загрузите изображение, выберите его, нажмите "Select".

Шаг 1.8. Публикация новой фотографии.

После загрузки и выбора нужной фотографии слайда нажмите "Publish" (Опубликовать), чтобы добавить новый слайд на сайт.

Удаление слайдов

Шаг 2.1. Удаление слайдов.

Чтобы удалить слайд, поставьте галочку рядом с ним, выберите "Move to Trash" (Переместить в корзину) в выпадающем меню и нажмите "Apply" (Применить).

Изменение контента

Шаг 3.1. Перейдите на страницу Home.

Для изменения контента главной секции Hero перейдите на страницу "Home" (Главная).

Шаг 3.2. Откройте Hero Section.

Найдите и разверните секцию "Hero Section", чтобы увидеть поля для изменения текста.

Шаг 3.3. Сохраните изменения.

После внесения изменений в текст нажмите кнопку "Save" (Сохранить) в правом верхнем углу, чтобы применить их на сайте.

Если вы хотите добавить или удалить пункт, зайдите в раздел "ACF" в боковом меню.

Выберите "Scopul programului" из списка.

Шаг 5. Добавление или удаление пунктов.

Здесь вы можете добавить новый пункт "contribuind" или удалить существующий.

Шаг 6. Следите за нумерацией.

Обязательно учитывайте порядок по цифрам в конце названия поля.

Шаг 7. Пример правильного заполнения.

Вот пример, как правильно и куда писать `contribuind`.

Шаг 8. Добавление или удаление.

Нажмите "Add Field" (Добавить поле), чтобы добавить новый `contribuind`, или "Delete" (Удалить), чтобы удалить существующий.

Шаг 9. Сохранение изменений.

После добавления нового поля вам нужно заполнить его значением и нажать "Save Changes" (Сохранить изменения), чтобы изменения применились на сайте.

Шаг 10. Изменения новых пунктов.

Руководство для разработчика

Это проект MDED. Ниже инструкция, как развернуть его на другом хостинге.

1. Требования к серверу

  • PHP 8.1+
  • MySQL / MariaDB
  • Apache или Nginx
  • phpMyAdmin или доступ к MySQL

2. Файлы проекта

1. Залей папку htdocs на сервер в директорию сайта (например: /var/www/html/ или в папку для домена).

3. База данных

  1. Зайди в phpMyAdmin.
  2. В существующей базе данных выбери «Импорт».
  3. Загрузи файл dump.sql (мой экспорт).
  4. В базе появятся таблицы проекта.

4. Конфигурация

  1. Найди файл конфигурации wp-config.php.
  2. Пропиши правильные данные для подключения к базе:
    • DB_NAME = имя твоей базы (уже существующей)
    • DB_USER = пользователь
    • DB_PASSWORD = пароль
    • DB_HOST = адрес у хостинга

5. Домен (опционально)

  1. Привяжи домен к папке htdocs.
  2. Если сайт «цепляется» к старому адресу, обнови ссылки (например, через таблицу wp_options: siteurl и home).

6. Плагины

1. Удостоверьтесь, что вы активировали следующие плагины:

  • Advanced Custom Fields
  • Yoast SEO
  • ACF Content Analysis for Yoast SEO
  • Polylang
  • TailPress

7. Тема

1. Активируйте тему MDED (фото синий фон белые буквы)

8. Готово 🎉

После этих шагов сайт MDED должен активироваться для использования.
